### Kleine Markdown Anleitung

Markdown ist eine einfache Syntax um Texte fürs Web zu erstellen. Hier ein paar Beispiele:

```markdown
# Titel

- Einfache
- Liste

1. Nummerierte
2. Liste

**Fett** and _Schräg_ and `Code` text

[Link](url) and ![Image](src)
```
